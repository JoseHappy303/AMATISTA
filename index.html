Sobre Nosotros

Amatista Artx

El sistema de "Amatista Artx", se puede encontrar pinturas de diferentes temas. Este sitio estará alojado en la web, para que cualquier persona interesada en adquirir uno de estos productos pueda consultarlos en cualquier momento, con el simple hecho de entrar a la página web.

El sitio contará con productos actualizados en todo momento, junto con una pasarela de pago para realizar compras en internet. La información del usuario será almacenada en una base de datos, para llevar un registro del usuario, y de los productos que han sido vendidos, para asegurar la existencia de estos.
